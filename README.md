# UIverse üåå

**The Ultimate Modern UI Playground.**

Built with **React**, **Vite**, **TypeScript**, and **Tailwind CSS**. Designed to help developers experiment with, customize, and copy-paste beautiful UI components.

![UIverse Hero](https://images.unsplash.com/photo-1550745165-9bc0b252726f?auto=format&fit=crop&q=80&w=2070)

## üöÄ Features

### Core
- **Authentication**: Secure Sign In/Sign Out powered by **Supabase**.
- **User Profiles**: Manage display names and settings.
- **Responsive Layout**: Validated across device sizes (Mobile audit pending).
- **Dark Mode**: Button toggle with smooth system-wide transitions.

### Interactive Playground üéÆ
The heart of UIverse. Tweak values and get production-ready code instantly.
1.  **Card**: Rotate, Scale, and Glow effects.
2.  **Button**: customize `variant` (destructive, outline, ghost) and `size`.
3.  **Input**: Test toggles for `type` (text/password), `placeholder`, and `disabled` state.
4.  **Toast**: Configure and trigger dynamic toast notifications.
5.  **Dialog**: Customize and open modal dialogs.
6.  **Code Copy**: One-click copy for generic JSX/CSS.

### Completeness
- **Global Footer**: Consistent branding and links.
- **Custom 404 Page**: Stylized error handling with "Ghost" animation.
- **Documentation**: "Getting Started" and "Components" guides.

## üèÉ‚Äç‚ôÇÔ∏è Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/ShridharB-cloud/Ui-verse.git
    cd Ui-verse
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root directory:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
